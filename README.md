# BlogManagerBackend

## Requiremnts
    Java JDK 1.8
    SpringBoot version : 2.3.2
    maven : 3.6

#### Using Eclipse
    Import the maven projet to eclipse
    Right click on project: Maven install
    Right click on project: Run as Spring Boot Application
#### Using cmd
    open up a cmd window at the project folder
    mvn spring-boot:run    

## ALL Done!
